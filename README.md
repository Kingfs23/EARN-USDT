# USDT Mine Landing Page

This project is now a pure static site.

You can:

- open it directly on your computer
- upload it to GitHub
- deploy it on Vercel without a backend

## Main files

- `index.html`
- `styles.css`
- `app.js`
- `favicon.svg`
- `logo-mark.svg`
- `usdt-coin.svg`
- `vercel.json`

## Local preview

Open `index.html` in your browser.

## Deploy on Vercel

1. Push this folder to GitHub.
2. Import the repository into Vercel.
3. Deploy it as a static site.

## Notes

- The page includes animated background effects and animated USDT hero motion.
- The form behavior is front-end only and meant for demo presentation.
